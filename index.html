即時報價（若能載入）來自臺灣證券交易所盤中資訊；若無法載入即時報價（常見於瀏覽器安全限制），會自動改用每日收盤資料，由 GitHub Actions 每天定期自動抓取並存進這個 repo，卡片上會標示「昨收」與資料更新時間。
輸入框可以打股票代號或中文名稱（例如「台積電」），名稱查詢只涵蓋上市股票，上櫃股票請直接輸入代號。
每張卡片可展開 TradingView 技術分析圖表（K線＋均線／RSI／成交量，可自行調整指標），資料約有 15 分鐘延遲；上櫃股票的交易所前綴是用猜的，若圖表顯示查無此股，代表猜錯，之後可以再手動調整。
每張卡片下方會顯示 相關新聞（由 Google 新聞彙整經濟日報、工商時報等多家來源，每日自動更新數次），只涵蓋 repo 裡 watchlist.json 名單中的股票，要新增股票的新聞，需要另外編輯那個檔案。
右側會顯示 獲利快照（營業毛利／營業利益／稅前淨利／本期淨利／累計EPS），資料來自證交所公開的季報，每週更新一次（財報本來就是季更，不是天天變），目前只涵蓋「一般業」，金融/證券/保險/金控類股暫不支援。
每檔卡片下方的「分析筆記」欄位可手動貼上你跟 Claude 聊天分析出的合理股價區間 / 技術型態判讀，方便對照。
本頁面僅供個人資訊整理使用，不構成投資建議。
