設定好之後，之後在網頁上「加入觀察」的股票會自動同步到 GitHub 的 watchlist.json，並自動觸發新聞／股價／獲利的抓取排程，不用再手動去 GitHub 操作。金鑰只會存在你這台裝置的瀏覽器裡，不會寫進公開的網站程式碼。

沒有 Token？點這裡看申請步驟

1. 打開 github.com/settings/personal-access-tokens/new
2. Repository access 選「Only select repositories」，選你的儀表板 repo
3. Permissions 展開 → Contents 設為 Read and write、Actions 設為 Read and write
4. 拉到最下面按 Generate token，把產生的字串（ghp_ 開頭）貼到上面欄位

或者不想設定金鑰，也可以繼續用手動複製貼上的方式：

載入中…
正在讀取股票基礎資料
即時報價（若能載入）來自臺灣證券交易所盤中資訊；若無法載入即時報價（常見於瀏覽器安全限制），會自動改用每日收盤資料，由 GitHub Actions 每天定期自動抓取並存進這個 repo，卡片上會標示「昨收」與資料更新時間。
輸入框可以打股票代號或中文名稱（例如「台積電」），名稱查詢只涵蓋上市股票，上櫃股票請直接輸入代號。
每張卡片右側會直接顯示 股價K線走勢圖（近半年，含成交量），資料來自 FinMind 官方股價歷史，畫圖用 TradingView 開源的 Lightweight Charts 函式庫（不經過他們的即時資料源，所以不會有登入限制的問題），每天收盤後自動更新一次，非即時。
每張卡片下方會顯示 相關新聞（由 Google 新聞彙整經濟日報、工商時報等多家來源，每日自動更新數次），只涵蓋 repo 裡 watchlist.json 名單中的股票，要新增股票的新聞，需要另外編輯那個檔案。
右側會顯示 獲利快照（營業毛利／營業利益／稅前淨利／本期淨利／累計EPS），資料來自證交所公開的季報，每週更新一次（財報本來就是季更，不是天天變），目前只涵蓋「一般業」，金融/證券/保險/金控類股暫不支援。
每檔卡片下方的「分析筆記」欄位可手動貼上你跟 Claude 聊天分析出的合理股價區間 / 技術型態判讀，方便對照。
